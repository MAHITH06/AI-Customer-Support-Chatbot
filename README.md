# AI-Customer-Support-Chatbot
Developed an AI-powered customer support chatbot using Python, Flask, and the OpenAI API to automate responses to user queries. Built RESTful APIs, applied prompt engineering and conversational memory for context-aware interactions, and designed a responsive web interface using HTML, CSS, and JavaScript.

- AI-powered customer support chatbot
- Natural language conversation
- Prompt engineering
- Conversation memory
- REST API integration
- Responsive web interface
- Chat history
